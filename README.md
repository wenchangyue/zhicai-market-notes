# 智财小站｜美股趋势交易笔记

智财小站的公开 companion site，用 GitHub Pages 托管校对后的公开视频逐字稿、简要回答、常见问题、来源和方法说明。

- 站点：<https://wenchangyue.github.io/zhicai-market-notes/>
- YouTube：<https://www.youtube.com/channel/UCJzXJ6OJCOLtK4OvHvjd7EA>
- 语言：简体中文

## 发布一条 companion page

1. 复制 `TEMPLATE-episode.md` 到 `_episodes/<slug>.md`。
2. 填写公开视频 ID、日期、ISO 8601 时长、简要回答、FAQ 和来源。
3. 粘贴与最终中文字幕一致的校对逐字稿。
4. 本地或 CI 检查 JSON-LD、链接、`robots.txt`、`sitemap.xml` 和移动端布局。
5. 提交到 `main`，等待 GitHub Pages 构建。
6. 把 companion page 链接加入对应 YouTube 视频简介。

尚未公开的长视频不建立可索引页面，也不把 unlisted URL 写入本站。
